NZ Volcanoes and World Population Web Map

The map has three layers:

Layer 1: base map with location names, roads and borders. Layer 2: polygon layer displaying color-coded population size (from green to red). Layer 3: Markers displaying the location of NZ volcanoes with a popup window with its names. On the top right-hand corner there's a layer control device that gives the user the ability to turn individual layers on and off. Roll the Dice

Game rules:

• A browser-based game for 2 players. In each turn, the players roll the dice as many times as they wish. Each result gets added to a round score. If the players roll a ‘1’, all round score points get lost and the turn is passed to the next player. Players can choose to 'Hold' their round score global score and pass the turn. The first player to reach 50 points first wins the game. Tools: HTML, CSS, Javascript.
